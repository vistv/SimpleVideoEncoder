Aim

Simple video encoder is frontend for ffmpeg designed to encode video files. The application is designed so that the process of starting the encoding of one or more videofiles takes 2-3 clicks.


Working with the application

1. To encode one or more videos, select them using the “Choose file(s)” button to select some video files or use the “Choose directory” button to select all video files in a specific directory.

2. Set the encoding parameters.

3. Press the “Encode” button to start encoding.


Data output

The encoded video will be placed in the “conv” directory, which will be created in the directory, where the original videofile is located. The original video file will be moved to the “done” directory.


Encoding parameters

1. On the front panel of the application you can select the resolution of the final video and bitrate.

2. You can select one-pass or two-pass encoding in the settings.

3. You can also set the encoding criterion in the style of "encode only those videos in which the hour duration is larger than xxx MB", which is useful if the recoding is done to reduce the video size.

4. The “Use own encoding parameters” option provides ample opportunities for fine-tuning of the encoding. Here it is possible to set features of encoding using parameters of ffmpeg (see ffmpeg documentation). By default, the program uses the following commands for encoding.

One-pass recoding:

ffmpeg.exe -y -i InputFile -c: v libx264 -f mp4 -b: v btrtK -vf scale = x: y OutputFile

Two-pass recoding:

ffmpeg.exe -y -i InputFile -c: v libx264 -f mp4 -b: v btrtK -vf -pass 1 scale = x:y NULL

ffmpeg.exe -y -i InputFile -c: v libx264 -f mp4 -b: v btrtK -vf -pass 2 scale = x:y OutputFile,

where btrtK - bitrate in kBps; x:y - resolution of video.

If you use the “Use own encoding parameters” option, the parameters highlighted in red will be replaced by user-defined parameters.


Most Common Video Resolutions are the following ones:

  • 352 x 240 (240p) (SD)
  • 480 x 360 (360p)
  • 858 x 480 (480p)
  • 1280 x 720 (720p) (Half HD)
  • 1920 x 1080 (1080p) (Full HD)
  • 3860 x 2160 (2160p) (Ultra-HD) (4K)

Bitrates (kBps) are calculated as follows:

Middle bitrate: mid_bitrate = (2 * resol_x * resol_y + 180000) / 1000

Low bitrate: low_bitrate = mid_bitrate / 2

High bitrate: high_bitrate = mid_bitrate * 2
